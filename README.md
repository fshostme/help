# FSHOST Help Documentation

Official help documentation for [FSHOST.me](https://fshost.me) - Free and Pro Game Server Hosting.

## Development

### Prerequisites

- Node.js (v18 or higher)
- npm

### Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run docs:dev
   ```

4. Open your browser to `http://localhost:5173` or whatever is shown

### Build Commands

```bash
# Start development server with hot reload
npm run docs:dev

# Build for production
npm run docs:build

# Preview production build locally
npm run docs:preview
```
