# Troubleshooting

Should you come across an issue, check here for troubleshooting with your FSHOST.me game servers.

## Most Common Problems

### Server Issues

- **[Connection Problems](/troubleshooting/connection)** - Can't connect to server

### Network Issues

- **High Ping** - Latency and connection quality
- **Timeout Errors** - Connection timeouts

## Quick Troubleshooting

### When Contacting Support

- **[Pro Support](https://fshost.me/pro/support)** - Follow the options on the Support page on our website

Include this information:

1. **Problem Description**
   - What's not working?
   - When did it start?
   - What changed recently?

2. **Error Messages**
   - Screenshots
   - No need to send logs as we can access them

3. **Steps Tried**
   - What have you already attempted?
   - What were the results?
