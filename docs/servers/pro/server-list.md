# Pro Server List

Your FSHOST.me Pro Server List is the central hub for managing all your game servers.

## Accessing the Server List

After logging in, you can find your servers via the 'server' icon next to your balance at **[https://fshost.me/pro/servers](https://fshost.me/pro/servers)**.

## Sections

### Server List

The main area shows all your servers:

- **Server ID** - ID which is used to identify it
- **Location** - Server region and which server ID if hovered on
- **Game** - Game name
- **Description** - Shown if entered in edit server page
- **Status** - Running, Stopped, Updating, or Crashed
- **Edit/Setup** - Button to edit the server settings
- **View** - Button to view the server and control it
- **View more arrow** - See quick connect command and price/renewal date
