# Config Files

Load predefined settings using the `exec` command:

```
rcon exec filename
```

## Available Configs

| Filename | Info |
| --- | --- |
| `casual.cfg` | Setting gamemode to Casual mode |
| `comp.cfg` | Setting gamemode to default competitive |
| `cheats.cfg` | To enable `sv_cheats 1` |
| `dm.cfg` | To enable deathmatch-plugin |
| `dz.cfg` | Setting gamemode to Dangerzone mode |

### Plugin Configs

| Filename | Info |
| --- | --- |
| `pracc.cfg` | **Pro servers only.** Custom saves for nades/bots/replays. See [splewis/csgo-practice-mode](https://github.com/splewis/csgo-practice-mode#commands) for available commands. |
| `pug.cfg` | See [splewis/csgo-pug-setup](https://github.com/splewis/csgo-pug-setup#commands) for available commands. Make sure to reload the config after the map veto. |
| `retakes.cfg` | To change weapons type `!guns` into chat |
| `scrim.cfg` | Standard scrim config with all rounds enabled. |
| `warmod.cfg` | See [alliedmods forum](https://forums.alliedmods.net/showthread.php?t=225474) for available commands. |
