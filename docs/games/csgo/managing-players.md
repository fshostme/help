# Managing Players

## Status

Shows the Slot Number, Name, SteamID, IP, and other info of the players connected.

```
rcon status
```

## Player List

Get a list of connected players:

```
rcon users
```

## Kick

Kick a player by their ID:

```
rcon kick id
```

## Unban

Unban a player by ID:

```
rcon removeid 1
```

Unban a player by IP:

```
rcon removeip 1
```

Unban all IDs:

```
rcon removeallids
```
