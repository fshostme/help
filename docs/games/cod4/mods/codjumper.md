# CoDJumper v3.12b

Bounce and jump modification for parkour-style gameplay.

## Overview

CoDJumper is a specialized mod created by [CoDJumper.com](https://www.codjumper.com/) that transforms Call of Duty 4 into a parkour and movement challenge game. Players navigate custom-built jump maps using advanced movement techniques.

## What is CoDJumper?

### Core Concept

CoDJumper focuses on:
- Movement mechanics mastery
- Jump technique practice
- Map navigation challenges
- Time trial competitions
- Skill-based progression

### Gameplay Style

**Not combat-focused:**
- No traditional shooting gameplay
- Focus on movement and jumps
- Parkour-style challenges
- Precision platforming

**Skill-based:**
- Master bounce mechanics
- Learn strafe techniques
- Perfect timing required
- Progressive difficulty

## Playing CoDJumper

### Getting Started

**Basic controls:**
1. Jump (space)
2. Crouch (C)
3. Strafe (A/D)
4. Forward (W)
5. Sprint (Shift)

**First steps:**
- Start on easy maps
- Practice basic jumps
- Learn strafe jumping
- Watch experienced players

### Advanced Techniques

**Strafe jumping:**
- Build speed through air strafing
- Maintain momentum
- Chain jumps together
- Navigate long distances

**Bounce jumps:**
- Precise timing required
- Height-gaining technique
- Used for vertical climbs
- Essential for hard routes

**Wall runs:**
- Maintain speed on angled surfaces
- Chain between walls
- Access secret areas
- Advanced technique

## Server Setup

### For Server Owners

**Running CoDJumper:**
1. Select CoDJumper mod when creating server
2. Choose appropriate maps
3. Configure save system
4. Set up timing
5. Enable spectator mode

**Recommended settings:**
- Friendly fire: Off
- Killcam: Off
- Fall damage: Configured per map
- Spawn protection: Off

## Troubleshooting

- Try easier maps
- Ask community for tips
:::

::: details Too difficult

- Join practice servers
