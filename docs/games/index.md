## Request a Game

If we don't currently support your desired game:

1. Join our **[Discord](https://fshost.me/discord)**
2. Make a suggestion in #free-support or create a Pro ticket
3. We are happy to add new games based on demand (if we can support it)

### Check Current Support

For the most up-to-date list of supported games:

1. Visit **[fshost.me/games](https://fshost.me/games)**
2. Click on your desired game and check we support free/pro for it
